# twpic
Twitterでフォローしている絵師の投稿画像を新着順で自動保存するツールです。

1.twipicディレクトリに[config.py]で自分のTwitterディベロッパーアカウントのユーザー情報を入れてください。
Linux 例）/home/[ユーザー名]/tweepy/twpic/config.py
Win 例）C:\\tweepy\\twpic\\config.py

'''python:config.py
consumer_key = 'ココ'
consumer_secret = 'ココ'
access_token = 'ココ'
access_token_secret = 'ココ'
'''
